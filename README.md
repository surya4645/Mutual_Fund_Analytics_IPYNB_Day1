# Mutual Fund Analytics — Day 1 IPYNB Project

This project is prepared for Bluestock Fintech Day 1 task: **Project Setup + Data Ingestion (ETL)**.

## How to Run in VS Code

1. Open this folder in VS Code.
2. Install Jupyter extension if not installed.
3. Open `notebooks/Day1_ETL_Report.ipynb`.
4. Select Python kernel.
5. Run all cells from top to bottom.

## Output Generated

After running the notebook, you will get:
- `data/raw/live_nav_data.csv`
- `data/processed/*_processed.csv`
- `reports/day1_data_quality_summary.txt`

## Folder Structure

```text
data/raw/          original datasets
data/processed/    processed output files
notebooks/         Jupyter notebook report
reports/           data quality reports
sql/               reserved for SQL work
dashboard/         reserved for dashboard work
```
